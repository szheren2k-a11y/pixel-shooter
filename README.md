# pixel-shooter

一个单文件实现的像素风自动射击小游戏，核心玩法围绕 **波次战斗、BOSS 掉落、装备养成、仆从辅助、扫荡跳关、转生加成** 展开。

## 玩法简介

- 鼠标移动角色，角色会自动朝前持续射击。
- 每 5 波出现一个 BOSS。
- 击败敌人可获得金币、分数，并有概率掉落装备。
- 可在商店中购买战斗升级，在仆从页招募辅助单位，在装备页更换装备并激活套装效果。
- 通关过的 BOSS 会解锁扫荡与开局快速跳关。
- 累计分数达到要求后可以转生，获得永久收益加成。

## 当前功能

### 战斗系统
- 自动射击
- 散弹、穿透、导弹等武器强化
- 生命、护盾、护甲、偏转、冲刺等防御成长
- 普通波次与 BOSS 波次
- 多种敌人弹幕模式

### 仆从系统
- 弓箭手：追踪连射
- 骑士：定期叠护盾
- 法师：范围魔法弹
- 牧师：自动回血

### 装备系统
- 四个装备槽位：武器、护甲、头盔、饰品
- 五档稀有度：普通、优秀、精良、史诗、传说
- 随机词条装备掉落
- 自动卖出低品质装备
- 三合一自动合成更高品质装备
- 套装效果：爆炸套、回血套、护盾套、暴击套

### 长线成长
- BOSS 通关进度记录
- 扫荡（x1 / x10）
- 开局快速跳关
- 转生永久收益加成

## 操作方式

- **鼠标移动**：控制角色位置
- **自动射击**：无需点击
- **空格**：冲刺（需先在商店解锁）
- **标签页**：切换游戏 / 仆从 / 装备 / 商店
