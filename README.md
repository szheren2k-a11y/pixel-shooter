# pixel-shooter

一个单文件实现的像素风自动射击小游戏，核心玩法围绕 **波次战斗、BOSS 掉落、装备养成、仆从辅助、扫荡跳关、转生加成** 展开。

## 玩法简介

- 鼠标移动角色，角色会自动朝前持续射击。
- 每 5 波出现一个 BOSS。
- 击败敌人可获得金币、分数，并有概率掉落装备。
- 可在商店中购买战斗升级，在仆从页招募辅助单位，在装备页更换装备并激活套装效果。
- 通关过的 BOSS 会解锁扫荡与开局快速跳关。
- 累计分数达到要求后可以转生，获得永久收益加成。

## 当前功能

### 战斗系统
- 自动射击
- 散弹、穿透、导弹等武器强化
- 生命、护盾、护甲、偏转、冲刺等防御成长
- 普通波次与 BOSS 波次
- 多种敌人弹幕模式

### 仆从系统
- 弓箭手：追踪连射
- 骑士：定期叠护盾
- 法师：范围魔法弹
- 牧师：自动回血

### 装备系统
- 四个装备槽位：武器、护甲、头盔、饰品
- 五档稀有度：普通、优秀、精良、史诗、传说
- 随机词条装备掉落
- 自动卖出低品质装备
- 三合一自动合成更高品质装备
- 套装效果：爆炸套、回血套、护盾套、暴击套

### 长线成长
- BOSS 通关进度记录
- 扫荡（x1 / x10）
- 开局快速跳关
- 转生永久收益加成

## 操作方式

- **鼠标移动**：控制角色位置
- **自动射击**：无需点击
- **空格**：冲刺（需先在商店解锁）
- **标签页**：切换游戏 / 仆从 / 装备 / 商店

## 本次修复

- 继续排查了“开始战斗后像是无限冲刺、无法正常玩”的问题，修复范围不只限于空格输入。
- 现在冲刺仍然基于按键状态单次触发，并在开始新一局和窗口失焦时重置空格状态，避免残留输入影响战斗。
- 额外加入了游戏运行实例标识：每次开始新游戏都会生成新的运行编号，旧的 `loop` / `tickSkip` 动画帧会自动失效，避免重复启动游戏后出现多条主循环同时运行、导致表现异常。
- 开始游戏时还会主动取消当前聚焦元素，减少按钮焦点残留带来的键盘干扰。

## 运行方式

直接用浏览器打开 `pixel_shooter_v5.html` 即可运行。
