# Pixel Shooter

一个单文件实现的像素风自动射击网页小游戏。当前主文件是 `pixel-shooter.html`，不依赖打包工具，直接用浏览器打开就能运行。

## 玩法概览

- 鼠标或手指移动角色，角色会自动射击。
- 每 5 波出现一个 BOSS。
- 击败敌人会掉落金币、装备，并有小概率掉落临时效果物品。
- 角色升级会获得技能点，可以在技能页购买永久技能。
- 仆从、装备、技能都会影响后续战斗强度。
- 只有转生时才会重置人物等级和技能等级。

## 当前系统

### 战斗

- 自动射击，支持速射、步枪、散弹、火炮、激光等武器子类型。
- 护甲套装和武器套装会提供额外效果。
- 每过 1 个 BOSS 阶段，怪物生命和伤害会大幅提升。
- 怪物移动速度、出场速度、子弹速度固定，不随阶段暴涨。
- 击杀怪物后有小概率掉落临时效果：
- 10 秒经验加倍
- 10 秒金币加倍
- 1 次护盾
- 全屏清除敌方子弹

### 技能

- 技能页通过技能点购买，不再使用战斗内三选一卡牌。
- 每升 1 级获得 1 点技能点。
- 技能只在转生时清空，死亡不会清空。
- `磁铁 / 多重射击 / 穿透 / 暴击 / 护盾` 最多购买 5 次。
- 以上有限技能的价格是 `10 -> 100 -> 1000 -> ...`。
- 其他无限技能的价格是 `1 -> 2 -> 4 -> 8 -> ...`。

### 装备

- 装备槽位包括：武器、护甲、头盔、饰品。
- 装备有普通、优秀、精良、史诗、传说五个品质。
- 装备会随机生成属性、护甲套装、武器套装和武器子类型。
- `强化` 会提高当前装备数值。
- `升阶重铸` 会消耗 1 件同部位同品质装备，并随机刷新属性和套装效果，不提升品质。
- 支持自动卖出低品质装备和三合一自动合成。

### 仆从

- 仆从包括弓箭手、骑士、法师、牧师。
- 金币强化只提升仆从数值，不提升机制效果。
- 碎片升阶会提升仆从效果，所有品质都可以升阶。
- 碎片需求是 `1 -> 5 -> 25 -> 125 -> ...`。
- 效果成长方向：
- 弓箭手：增加弹道
- 骑士：减少护盾冷却时间
- 法师：增加爆炸范围
- 牧师：增加回复速度

### 转生

- 转生币获取公式：`击败的 BOSS 数量^3 + 当前人物等级`。
- 转生会清空人物等级、技能等级、已穿戴装备和背包。
- 已通关波次记录会重置。

## 操作方式

- 桌面端：移动鼠标控制角色。
- 移动端：拖动手指控制角色。
- 角色会自动攻击，无需额外点击。
- 通过顶部页签切换 `游戏 / 仆从 / 装备 / 技能`。

## 运行方式

直接用浏览器打开 `pixel-shooter.html`。

如果你想用本地服务启动，也可以在仓库目录执行任意静态文件服务，例如：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000/pixel-shooter.html`。

## 开发说明

- 这是一个纯前端单文件项目，HTML、CSS、JavaScript 写在同一个文件里。
- 适合快速迭代玩法、数值和平衡性。
- 如果要继续扩展，优先建议把数值配置、掉落表、技能定义、仆从定义拆成独立常量区域，方便后续维护。
